# FPS Guess

左右のアニメーションを見比べて、どちらがなめらかかを当てるブラウザゲームです。

## 遊び方

1. 下限と上限のFPSを設定します。
2. 左右の動きを見比べます。
3. 「左」「右」「同じ」から、よりなめらかな方を選びます。
4. 10問終わると最終スコアが表示されます。

## 公開方法

このゲームは静的ファイルだけで動くため、GitHub Pagesでそのまま公開できます。

1. GitHubで新しいリポジトリを作成します。
2. このフォルダをリポジトリにpushします。
3. GitHubの `Settings` -> `Pages` で、`main` ブランチのルートを公開元に設定します。

## ファイル構成

- `index.html`: 画面構造
- `style.css`: 見た目
- `script.js`: ゲーム処理
